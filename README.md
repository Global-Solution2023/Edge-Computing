<h1>Global Solution - Controlador de temperatura e umidade</h1>

<h2>Descrição do programa</h2>

Protótipo do projeto de um controlador de temperatura e umidade, que ligado por meio de IoT com as tecnologias de secagem e resfriamento, além da aplicação web, controlariam os grãos para se manterem nas condições ideais

<h2>Materiais necessários</h2>

1x Placa Arduino UNO R3;<br>
1x Cabo USB;<br>
1x Breadboard;<br>
1x Soil Moisture Sensor;<br>
1x Display LCD 16x2; <br>
1x Temperature Sensor;<br>
1x Potenciometro; <br>
22x Cabos jumper ;<br>
1x Resistores (1 k ohms) ;<br>

<h2> Funcionalidades </h2>
<li>Medir a temperatura dos grãos
<li>Medir a umidade dos grãos

<h2>Como reproduzir</h2>

<ol>
    <li>Incluir a biblioteca
    <li>Declarar as variáveis e os pinos
    <li>Defina os pinos de entrada e saída
    <li>Crie as funções, uma para a parte de temperatura e outra para a parte de umidade
    <li>Coloque as funções em looping
    </ol>
